TP de IA - desenvolvimento de aplicação com inteligência artificial.

Esta é uma aplicação desenvolvida por Felipe da Silva Reis e Guilherme de Mello, com uso de uma biblioteca de machine learning para javascript a brain.js

A aplicação resulta qual é a categoria da sua frase digitada, as categorias registradas até o momento para o aprendizado da aplicação são software e hardware, portanto a aplicação retornara uma dessas categorias baseada na frase digitada. Dependendo de sua confirmação a aplicação irá guardar os dados registrados no arquivo de aprendizagem.

A aplicação ainda esta em treinamento, pode demorar para a visualização da resposta.
